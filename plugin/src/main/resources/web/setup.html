MCStatsBot Setup

Richte deinen Discord-Bot in wenigen Schritten ein

1. Discord Bot erstellen

Erstelle eine Discord-Anwendung und kopiere den Bot-Token.

  1. Oeffne das Discord Developer Portal
  2. Klicke New Application und gib einen Namen ein (z.B. "MCStatsBot")
  3. Gehe links auf Bot
  4. Klicke Reset Token und kopiere den Token
✓

2. Bot zum Server einladen

Klicke den Button um den Bot zu deinem Discord-Server hinzuzufuegen.

Bot zum Server hinzufuegen

Nachdem du den Bot hinzugefuegt hast, klicke auf "Weiter".

Tipp: Du kannst den Bot zu mehreren Servern hinzufuegen.

3. Server & Channel waehlen

Waehle den Server und Channel fuer die taeglichen Reports.

✓

Setup abgeschlossen!

Dein Discord-Bot ist jetzt aktiv und verbunden.
